# Welcome to the BubbleBubble Project
A full-stack Django e-commerce application built as part of a portfolio project.

BubbleBubble (rebranded as *Moon & Moss*) demonstrates a modular Django architecture, secure user authentication, shopping cart and checkout functionality, Stripe payment integration, and a custom owner dashboard for product and order management.
